1.3.3. kwrecord

Dieses script brennt Daten, Iso Images, Audio und Video-CDs auf CD-R(W) und DVD-+R(W). Die zugrundeliegende Funktion cd_dvd_burn kann man in jedem Script nutzen. Beispiel: cd_dvd_burn --iso, brennt ein oder mehrere ISO Images. Schauen Sie dazu in den Header der Funktion.

Notiz: Dieses Script benutzt die Programme cdrdao und growisofs.


1.3.3.0 Konfiguration

Hier kann man ein paar Brenn(er) Einstellungen taetigen.

Medium: Hier stellt man das Medium ein, auf welchen die Daten gebrannt werden sollen. Zur Auswahl stehen CD und DVD.

CD-Groesse: Hier stellt man die Groesse des Rohlings ein. Zur Auswahl stehen 650, 680, 700 und 800Mb.

Speed: Hier stellen Sie die Brenngeschwindigkeit fuer CD ein. Meistens steht auf der CD-R(W) Huelle die Geschwindigkeit drauf die diese vertragen. Welche Geschwindigkeit ihr Brenner unterstuetzt entnehmen Sie bitte ihrer Hardware Dokumentation.

DVD-Groesse: Hier stellt man die DVD Groesse des Rohlings ein. Zur Auswahl stehen 4,7, 9,4 und 15,9 Gb.

DVD-Speed: Hier stellen Sie die Brenngeschwindigkeit fuer DVD's ein. Entnehmen Sie dies bitte der Dokumentation zu Ihren DVD Brenner.

Brenner: Hier wird Ihnen eine Auswahl von Brennern angeboten, die in ihrem Rechner vorhanden sind.

Leselaufwerk: Hier wird Ihnen eine Auswahl von Leselaufwerken angeboten, die in ihrem Rechner vorhanden sind.

Treiber: Hier wird Ihnen eine Auswahl von Treibern angeboten, die cdrdao unterstuetzt. Wenn keiner fuer Ihren Brenner dabei ist, versuchen Sie generic-mmc oder generic-mmc-raw.

CD/DVD auswerfen: Soll die CD/DVD nach dem brennen ausgeworfen werden(yes/no)?

Simulieren: Soll der Brennvorgang nur simuliert werden (yes/no)?

Notiz: Es werden nur bis zu 10 Laufwerke unterstuetzt.



1.3.3.1 Audio

Nur Audio-Dateien werden zum brennen, zur Auswahl, angeboten.


1.3.3.2 Copy-CD

Hier koennen Sie eine 1 zu 1 Kopie brennen.


1.3.3.3 Daten

Hier koennen Sie Verzeichnisse oder einzelne Dateien brennen.


1.3.3.4 Iso

Nur Iso-Images werden zum brennen, zur Auswahl, angeboten.


1.3.3.5 Photo

Nur Bilder werden zum brennen, zur Auswahl, angeboten.


1.3.3.6 Video-CD-DVD

Brennen von Video-CD' und Video-DVD's. Es werden nur diese angeboten.


1.3.3.7 loeschen/formatieren

Hier kann man entweder eine CD+-RW loeschen oder eine DVD+-RW und DVD-Ram formatieren. Kommt drauf an was gerade in der Konfiguration steht (CD oder DVD).
